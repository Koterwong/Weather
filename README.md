## 介绍

外观设计高仿了小沃天气，天气数据来源百度API平台和风数据。主要包含以下模块。

- 天气界面
- 添加城市
- 管理城市
- 设置：定位，后台更新，通知栏，缓存
- 关于

##使用到的开源库

- okHttpUtils
- HelloCharts，          
- view-pager-transforms
- RxJava          
- swipeBackLayout
- ACache
